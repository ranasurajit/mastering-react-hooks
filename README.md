A **hands-on React learning journey** where we master React Hooks by building real-world applications progressively.

Instead of learning hooks individually, this repository focuses on learning them **in the context of real applications**, just like professional React development.

# 🧠 Learning Philosophy

Most tutorials teach hooks like this:

```
useState
useEffect
useContext
...
```

But real applications don't work that way.

This repository focuses on:

✔ **Problem → Hook → Solution**

You will learn:

* Why the hook exists
* When to use it
* How it improves architecture

---

# 🏗 Applications

All applications are organized inside:

```
applications/

Example:

```
applications/
   smart-task-manager/
   performance-lab/
   hooks-playground/

# 🗺 React Hooks Learning Roadmap

Track progress through the journey.

### 🟢 Core Hooks

* [ ] Step 1 — Build Todo Manager → `useState`
* [ ] Step 2 — Persist Tasks → `useEffect`
* [ ] Step 3 — DOM Access & Persistent Values → `useRef`

---

### 🟡 State Architecture

* [ ] Step 4 — Global Theme / User State → `useContext`
* [ ] Step 5 — Complex State Management → `useReducer`

---

### 🟠 Performance Optimization

* [ ] Step 6 — Optimize Expensive Computations → `useMemo`
* [ ] Step 7 — Prevent Unnecessary Re-renders → `useCallback`

---

### 🔵 Advanced Component Control

* [ ] Step 8 — Imperative Component APIs → `useImperativeHandle`
* [ ] Step 9 — DOM Measurements & Layout Updates → `useLayoutEffect`

---

### 🟣 Concurrent React

* [ ] Step 10 — Non-blocking UI Updates → `useTransition`
* [ ] Step 11 — Deferred Rendering → `useDeferredValue`

---

### 🔴 Advanced / Library Hooks

* [ ] Step 12 — External Store Synchronization → `useSyncExternalStore`
* [ ] Step 13 — Debugging Custom Hooks → `useDebugValue`
* [ ] Step 14 — CSS-in-JS Optimization → `useInsertionEffect`

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/react-hooks-mastery.git
```

Navigate to the repository

```bash
cd react-hooks-mastery
```

Run an application

```bash
cd applications/smart-task-manager
npm install
npm start
```
